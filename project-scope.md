# Ai-powered ticket management system

## problem

we recived hundreds of support emails dialy. our agents manully read, cassifu and respond to each tckets - which is slow and leads to impersonal ,canned resposes.

## solution 

build a ticket management system that uses AI automatically classsify, respond to and rout supprt tickets - delivering faster, more personalized responses to students while freeing up agents for complex issues.

## ticket statuses

- open — newly created, awaiting agent action
- resolved — a response has been sent or the issue addressed
- closed — no further action needed; ticket is archived

## ticket categories

- general question
- technical question
- refund request

## user roles

- admin — deployed with the system; can create and manage agent accounts
- agent — can view, respond to, and update tickets

## features 

- receive support emails and create tickets
- auto-generate human-friendly responses using a knowledge base
- ticket list with filtering and sorting
- ticket detail view
- AI-powered ticket classification (assigns category on creation)
- AI-suggested replies for agents to review before sending
- user management (admin only — create/manage agent accounts)
- dashboard to view and manage all tickets
